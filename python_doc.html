Basics

In this section, I'm going to show you some of the basics of Python to get you started on your first program!

Here are the concepts I will go over:

  1. print() method
  2. "strings"
  3. Operators (+, - , / , *)

print(): is a method in Python that allows you to print text in your console. Below I show an example of you would print the word "hello" to the console.

print("hello")

"Strings": are a text data type in Python that are not primitive. In Python, strings are characters that are surrounded by single quotes or double quotes. This syntax doesn't apply to all programming languages (such as C# in which characters are denoted with single quotes and strings are denoted with double quotes ;)). Strings don't just have to be made up of letters from the alphabet. See examples below:

type("hello") -> string (str) type("12345") -> string(str) type("12eg+") -> string(str)

Operators: are made up of simple operations that you would do in a math class. There are several types of operations you can do in Python such as addition, subtraction, division, and multiplication to name a few. Their symbols are just like the ones you would see in math class: + - / *

5+4 -> 9 6-10 -> -4 20/5 -> 4 100*2 -> 200
